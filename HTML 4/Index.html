Navegación Horizontal Perfecta

Una barra de navegación efectiva es el corazón de cualquier sitio web. Proporciona orientación, estructura y una experiencia de usuario fluida.

Elementos Clave

  • Lista de enlaces dentro de una etiqueta <nav>
  • Elementos alineados horizontalmente
  • Sin puntos de lista (eliminados con CSS)
  • Espacios uniformes entre botones
  • Sin subrayado predeterminado
  • Efectos hover para interactividad
  • Ocupa el 100% del ancho disponible

Código de la Navegación

HTML

<nav class="navegacion-principal">
  <ul class="menu">
    <li><a href="#inicio">Inicio</a></li>
    <li><a href="#servicios">Servicios</a></li>
    <li><a href="#portafolio">Portafolio</a></li>
    <li><a href="#nosotros">Nosotros</a></li>
  </ul>
</nav>

CSS

.menu {
  list-style: none; /* Elimina puntos */
  padding: 0;
  margin: 0;
  display: flex; /* Alineación horizontal */
  justify-content: space-around; /* Espacio uniforme */
}

.menu a {
  text-decoration: none; /* Sin subrayado */
  color: #333;
  padding: 15px 20px;
  display: block;
}

.menu a:hover {
  color: #3498db; /* Cambio de color en hover */
  background: #f5f5f5;
}
