¿Cómo funciona el Modelo de Caja?

Margin: Espacio externo que separa el elemento de otros elementos.

Border: Línea que rodea el padding y el contenido.

Padding: Espacio interno entre el contenido y el borde.

Content: El área donde se muestra el contenido (texto, imágenes, etc.).

Con box-sizing: border-box, el ancho y alto incluyen padding y border, ¡sin cálculos locos!
