Resumen de Requisitos Cumplidos

Redondeen las esquinas para un look moderno y suave

Implementado con border-radius desde 8px hasta 50px, usando variables CSS para consistencia.

Elijan un color de fondo que contraste con el diseño

Paleta de colores vibrantes que contrastan con fondos claros y oscuros, garantizando legibilidad.

Eliminen cualquier decoración de texto que traiga el link

text-decoration: none aplicado a todos los botones, eliminando subrayados por completo.

Añadan una transición para que el cambio de color sea fluido

transition: all 0.3s ease en todos los botones para animaciones suaves en hover y active.

Hagan que el cursor cambie a mano al posicionarse encima

cursor: pointer implementado en todos los estados interactivos de los botones.
